## Reflection

1. **What is Git?**
Tool that tracks changes in your code. Lets you save versions and go back if you break something.

2. **What is GitHub?**
Website to host your Git repos online. Lets others see your code and you can collab using PRs.

3. **Difference between Git and GitHub?**
Git runs on your machine, tracks history. GitHub is just where you host it online. Git doesn't need GitHub, but GitHub needs Git.

4. **Why are commits important?**
Each commit is a checkpoint. If you mess up later you can see what changed and undo it instead of starting over.

5. **Which Git command did you use the most?**
`git add` mostly, since you gotta stage files before every commit. Used `git status` a lot too just to check what's changed.
